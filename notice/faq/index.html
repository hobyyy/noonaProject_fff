  • 텀블벅 후원 쿠폰 사용 안내입니다. (스테디오 가입자는 해당되지 않습니다.)

    안녕하세요!  fourfortyfour 입니다!
    텀블벅 후원으로 쿠폰을 받으신 분들의 쿠폰 적용 방법에 대해서 설명해드리겠습니다!

    1. 먼저 스테디오 홈페이지의 우측 상단의 프로필 사진을 클릭해주세요.
    2. 그 뒤에 쿠폰함을 클릭해주세요.
    3. 메시지로 보내드린 쿠폰 번호를 입력 후 쿠폰 등록을 눌러주시면
    4. 보유쿠폰 1장이 생기게 됩니다. 우측 화살표를 눌러주세요.

    5. 이후 열리는 fourfortyfour 페이지에서 멤버십 가입 버튼을 눌러주세요.

    6. 첫번째 4:44 질문 멤버쉽(12,000원 항목)에 선택되어 있는 것을 확인해주시고,
    7. 아래에 후원 멤버십 가입 버튼을 클릭해주세요.

    8. 결제수단을 선택 해주세요. (※ 해당 멤버십은 1회성 멤버십으로 기간 종료 후 멤버십도 종료됩니다. 추가적인 결제는 없으므로 안심하시고 결제수단을 등록해주셔도 됩니다.)
    9. 이후 결제액 0원 을 확인 후 결제하기를 진행해주시면 됩니다.
    * 결제액 0원이 되어야 쿠폰이 적용된 상태입니다!
    결재액이 12,000원일시 다시 한 번만 더 확인해주세요!
